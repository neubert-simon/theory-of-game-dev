# Game Engines
> Arguably, a data-driven architecture is what makes the difference between game and game engine.Game engines should be extensible, so that they can be used to create many different types of games.

## What is a Game?
> Video Games are Soft Real-Time Simulations.

aka "Soft real-time interactive agent-based computer simulations".

**Computer simulation**
> A subset of a (imaginary) world is modeled mathematically so that it can be manipulated by a computer.

**Agent based simulation**
> A simulation in which number of distinct entities (agents) interact.

**Interactive real-time simulations**
> – The state changes over time (dynamic)<br>– Unpredictable player inputs (interactivity)<br>– Real-time response and feedback

**Soft real-time systems**
> A system in which missed deadlines are not catastrophic.

If the frame-rate dies, the human player generally doesn't!

## Deadlines
Games have numerous deadlines to meet
| Process | Duration |
| --- | --- |
| Rendering | ~30 frames / second |
| Physics | ~120 updates / second |
| Artificial Intelligence | ~think once per second |
| Audio Library | ~60 updates / seconds|

**Prioritization**
1. Audio / Physics
2. Rendering
3. AI

## Runtime Engine Architecture
- Game engines are built in layers
- Upper layers depend on lower layers
- Circular dependencies should be avoided

- **Rendering Engine** 
Processes code based assets and resources to enable displaying on the screen.

- **Skeletal Animation**
Animates skeleton of in game characters / objects. Bones are used for movement and calculations.

- **Collision & Physics**
Calculates game physics, i.e. the behavior of in-game objects.

- **HID layer**
Collects, possibly serializes and processes input from HIDs, and send output back to the devices. 

- **Platform Independence Layer**
Serves as middleware between the engine and the platform, abstracting implementation specific details like e.g. data types or file system information.

- **Online Multiplayer**
Handles online multiplayer tasks like matchmaking, game state replication etc.
